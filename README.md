# Ephermis Documentation HTML

This repository contains the Markdown for **Ephermis documentation**, designed to organise and present internal references and notes. It serves as a central hub for keeping track of project details, development progress, and related discussions.  

This repository also stores GitHub discussion threads for Ephermis projects, allowing developers to access, review, and analyse conversations and feedback in one place. This documentation's HTML is licensed under MIT (although Ephermis services and products are NOT licensed under MIT).
